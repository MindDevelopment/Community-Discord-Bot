# Changelog v1.0.0

**Release date:** 2026-07-23

---

## 🚀 Added

### Features
- Discord bot with slash command support (discord.js v14)
- Modular command handler with auto-loading from subdirectories
- Event handler with auto-loading
- SQLite database for persistent storage (warnings, suggestions, reports, settings)
- Command cooldown system
- Permission-based access control

### Commands
- **Info:** `/ping`, `/help`, `/botinfo`
- **Moderation:** `/kick`, `/ban`, `/clear`, `/timeout`, `/warn`, `/warnings`, `/unwarn`
- **Utility:** `/serverinfo`, `/userinfo`, `/avatar`
- **Fun:** `/8ball`, `/roll`
- **Community:** `/suggest`, `/report`

### Events
- `ready` — Sets bot presence and logs login
- `interactionCreate` — Routes slash command interactions
- `guildMemberAdd` — Sends welcome message

### Infrastructure
- Centralized configuration via `src/config.js` and `.env`
- Structured project layout (`src/commands/`, `src/events/`, `src/handlers/`, `src/utils/`, `src/database/`)
- Command deployment script (`src/scripts/deploy-commands.js`)
- Logging utility with severity levels
- Embed builder utility for consistent embeds
- Warning system with database persistence
- Suggestion system with reaction voting
- Report system for member reporting
